# Perl 数据类型

> Perl 是一种弱类型语言, 有三种变量类型的数据类型: 标量, 数组, 哈希. 还有一种就是直接量, 直接量包含: 整型, 浮点型, 字符串.

## 数据类型

| 类型 | 描述 | 示例 |
| :--- |:--- |:--- |
| 标量 | 用于表示一个直接量的变量 | $name='mirror'; $age=18; |
| 数组 | 用于标识一组直接量有序集合的变量 | @array=('a','b','c',1,2,3); |
| 哈希 | 用于标识一组(标量-直接量)的无序集合的变量 | %hash=('a'=>1,'b'=>2); |
