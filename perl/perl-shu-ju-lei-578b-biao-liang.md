## 比较运算符
perl 语言中, 字符串和数字的比较运算符是不同的, 这个需要搞清楚:

| 关系 | 数字比较 | 字符串比较 | 示例 |
| :--- |:--- |:--- | :--- |
| 相等 | == | eq | 1 == 2, 'a' == 'b' |
| 不等 | != | ne | 1 != 2 , 'a' != 'b' | 
| 大于 | > | gt | 1 > 2, 'a' > 'b' |
| 小于 | < | lt | 1 < 2, 'a' < 'b' |
| 不小于 | >= | gte | 1 >= 2, 'a' >= 'b' |
| 不大于 | < = | lte | 1 < = 2, 'a' < = 'b' |
