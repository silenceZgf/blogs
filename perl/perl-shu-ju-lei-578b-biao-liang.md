# Perl 数据类型--标量变量

> 标量是用于表示一个数字或字符串的变量, 一个标量只能表示一个.

# 1. 标量定义

标量使用$+名称表示

## 1.1 直接量定义

* $name = 'zhangsan';
* $age = 18;
* money = 10.9e5;

## 1.2 变量内插

* $hw = "hello, $name";
