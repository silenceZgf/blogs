# Shell 流程控制-[判断分支
> Shell 语言中用于判断的分支的语句有if-else 和 case. 


## 1. if-else
### 1. if-else 结构
```bash
# 第一种写法: 需要写; 要注意缩进  
if [ 条件判断表达式 ] ; then  
    # 程序块儿  
else   
    # 程序块儿  
fi  
      
# 第二中写法: 不需要写; 要注意缩进  
if [ 条件判断表达式 ]  
  then  
     #程序块儿  
else   
     #程序块儿  
fi  
```
